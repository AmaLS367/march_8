# Present

Vite + React + TypeScript project with shadcn-ui and Tailwind CSS.

## Setup

```sh
npm i
npm run dev
```

## Scripts

- `npm run dev` - start dev server
- `npm run build` - production build
- `npm run preview` - preview production build
- `npm run lint` - run ESLint
- `npm run test` - run tests

## GitHub Pages Auto Deploy

This repository is configured for automatic deploy to GitHub Pages via GitHub Actions:

1. Push to the `main` branch.
2. Open `Settings -> Pages` in GitHub and set `Source` to `GitHub Actions` (one-time setup).
3. Wait for workflow `Deploy to GitHub Pages` to finish.

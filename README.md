# Present

Vite + React + TypeScript project with shadcn-ui and Tailwind CSS.

## Setup

```sh
npm i
npm run dev
```

## Scripts

- `npm run dev` — start dev server
- `npm run build` — production build
- `npm run preview` — preview production build
- `npm run lint` — run ESLint
- `npm run test` — run tests

## Tech stack

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS
